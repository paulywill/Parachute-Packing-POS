== Parachute Packing POS

This is a Ruby on Rails POS I created for a local skydiving dropzone.

The parachute packers used this hacked up app to track their pack jobs.
One of my pro·té·gé took over the team and created another version that's CSS/PHP/MySQL with some invoice creation.

This was basically a beta version to get the job done.
